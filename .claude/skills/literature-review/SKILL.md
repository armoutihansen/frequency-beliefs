# Literature Review Skill

Use this skill when asked to investigate related literature, search online, summarize papers, or relate literature to the current project.

## Goal

Produce an accurate, useful literature map that helps position the paper.

## Procedure

1. Define the literature question.

2. Extract keywords from the current project:
   - belief elicitation
   - frequency reports
   - scoring rules
   - proper scoring rules
   - multinomial beliefs
   - count reports
   - exact-match scoring
   - quadratic scoring rule
   - risk aversion
   - belief identification
   - partial identification

3. Search in layers:
   - direct sources;
   - sources cited by direct sources;
   - foundational scoring-rule literature;
   - experimental-economics elicitation literature;
   - adjacent identification/design literatures.

4. For each source, record:
   - bibliographic information;
   - what was inspected: title/abstract/full paper/local PDF;
   - main contribution;
   - relation to current paper;
   - novelty risk;
   - how to use it;
   - confidence level.

5. Separate:
   - must cite;
   - probably cite;
   - useful background;
   - not relevant.

6. End with implications for the paper.

## Output format

Use:

- Search objective
- Search terms
- Must-cite sources
- Probably-cite sources
- Background sources
- Novelty risks
- Missing literatures
- Implications for framing
- Recommended next searches
- Suggested manuscript changes

## Rules

**Accuracy:**
- Do not invent citations or page numbers.
- Do not claim full-paper knowledge from a snippet.

**Communication:**
- Mark uncertainty clearly.
- Prefer accurate short summaries over broad vague coverage.
- If a source is local, say so.
- If a source was only found online, say so.
