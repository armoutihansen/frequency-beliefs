## Lens 2: Derivation Verification

### Issues

| # | Issue | Severity | Location |
|---|-------|----------|----------|
| D1 | Lemma `lem:manhattan-unimodal` strict-decrease claim. The proof argues `Σ_{v≠i} p_v^(L)(c)` is strictly decreasing in c to handle the `c < c*` case. This is strict only if at least one v ≠ i has r_v > 0 — which is guaranteed by r_i < n, as noted in A2 — but the proof doesn't connect those dots. If all v ≠ i had r_v = 0, the sum would be identically 0 and the c<c* argument would only give a weak inequality. The conclusion (φ attains max at c*) is still correct in that degenerate case because the max would be attained on a flat plateau ending at c*, but the lemma's "single monotone scalar equation" framing would be slightly off. The hypothesis r_i < n rules this out, so the proof is rigorous; a single sentence would make it airtight. **What would change my mind:** explicit note that r_i < n + Σr = n ⟹ some v ≠ i has r_v > 0, hence Σ_{v≠i} p_v^(L)(c) is strictly decreasing. | MINOR | 08_appendix.tex:320 |
| D2 | The D1 annotation fix in §3.2's appendix proof (08_appendix.tex:96-107) is correctly inserted: "since the remaining coordinates have r_v=0 and therefore contribute p_v≥0 to the simplex sum, Σ_{v≠i, r_v>0}p_v ≤ 1-p_i". This is mathematically tight: 1 = Σ_v p_v = p_i + Σ_{v≠i, r_v>0} p_v + Σ_{v: r_v=0} p_v, where the last sum is ≥ 0, giving Σ_{v≠i, r_v>0} p_v ≤ 1 − p_i. The inequality is exactly the one needed to close the previously-flagged gap. **What would change my mind:** N/A — this is correct. | (no issue) | 08_appendix.tex:96-107 |

### Positive Findings

- I re-verified the full envelope argument in Lemma `lem:manhattan-unimodal` part (i): φ(c) = min(p_i^(U)(c), 1 − Σ_{v≠i} p_v^(L)(c)); for c < c*, φ(c) = 1 − Σ_{v≠i} p_v^(L)(c) ≤ 1 − Σ_{v≠i} p_v^(L)(c*) = p_i^(U)(c*) by the weak decrease of Σ p_v^(L); for c > c*, φ(c) = p_i^(U)(c) ≤ p_i^(U)(c*). Both inequalities saturate at c*, so sup φ = p_i^(U)(c*). Argument is correct and tight.
- I re-verified the boundary values: f(0) = p_i^(U)(0) + Σ_{v≠i} p_v^(L)(0). With r_i < n, p_i^(U)(0) = 1 (unique p with F(r_i; n, p) = 0). For v ≠ i with r_v > 0, p_v^(L)(0) is unique p with F(r_v−1; n, p) = 0, so p_v^(L)(0) = 1. For r_v = 0, p_v^(L)(0) = 0 by convention. So f(0) = 1 + (m or m−1) ≥ 1. ✓ And f(1) = 0: p_i^(U)(1) = unique p with F(r_i; n, p) = 1, which is p = 0 (for r_i < n); each p_v^(L)(1) = 0 for r_v > 0 (unique p with F(r_v−1; n, p) = 1, p = 0). So f(1) = 0. ✓ Both boundary claims are correct.
- The squared-distance upper-bound proof (D1 fix) at 08_appendix.tex:96-107 is now self-contained and reads cleanly.
- Marginal-cost identity Δℓ_i(t; p_i) = 2 F_i(t; n, p_i) − 1 (08_appendix.tex:259-261) is correct and stable across the appendix and §3.4.
