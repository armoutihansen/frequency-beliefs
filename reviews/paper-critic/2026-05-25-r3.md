## Minor Issues

### m1 — Oxford-UK spelling regression in new C4 paragraph

**Category:** Consistency / LaTeX-Specific
**Location:** `paper/sections/07_discussion.tex:117`

**Quote (verbatim):**
> "Hedging-style behaviour aimed at balancing payoffs across realisations is structurally similar."

**Problem:** The paper is standardised on Oxford UK conventions: `-ize/-ization` combined with `-our/-re/-ce`. So `behaviour` is correct, but `realisations` is British `-ise` and conflicts with `realizations` and `realized` used in `01_introduction.tex:4`, `02_setup.tex:13, 38, 39, 50`, `03_scoring_rules.tex:49, 55, 137, 239`, `07_discussion.tex:83, 98`. This is the same class of regression that round 2 found in §3.4 (M2 then); the round-3 additions introduced one new instance.

**Fix:** `07_discussion.tex:117` — replace `realisations` with `realizations`. One-character change, no semantic effect.

### m2 — New C4 paragraph slightly overstates a cross-rule comparative claim

**Category:** Hedging / Style
**Location:** `paper/sections/07_discussion.tex:116`

**Quote (verbatim):**
> "This moves the squared-distance identified set the most, because that set is the projection polytope of the reported count vector and shifts continuously with the reported coordinates; for frequency-guessing the report is a multinomial mode, so discrete jumps in the report can leave the identified set unchanged across a range of true beliefs."

**Problem:** "The most" is a strict-maximum claim across three rules, but the surrounding paragraph only compares squared-distance to frequency-guessing explicitly, and the structural argument supports an ordering (squared-distance more sensitive than frequency-guessing, for the stated reason) rather than a strict-maximum over all three rules including Manhattan. The very next sentence then says "Hedging-style behaviour aimed at balancing payoffs across realisations is structurally similar." — confirming the claim is about a directional contrast with frequency-guessing, not a strict maximum. A careful referee will read "the most" as an unsupported strict-maximum and ask for a derivation.

**Fix:** `07_discussion.tex:116` — soften "moves the squared-distance identified set the most" to either "moves the squared-distance identified set more than the frequency-guessing identified set" (matches the pairwise structure of the argument) or "moves the squared-distance identified set continuously, in a way that the frequency-guessing identified set does not" (matches the mechanism sentence verbatim).
