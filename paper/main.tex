\title{The Informational Efficiency of Frequency-Report Scoring Rules}
\author{Jesper Armouti-Hansen}
\date{\today}

\begin{abstract}
In a multinomial frequency-report environment, a subject has latent beliefs \(p\), but the researcher observes only an incentivized count report \(r\).
We compare frequency-report scoring rules by how sharply that report identifies the latent beliefs.
The inference is recast as a partial-identification problem: each scoring rule maps the report to an identified set of beliefs---those under which the report would have been optimal---from which coordinate and mean bounds follow.
The main analytical result is that quadratic-distance frequency scoring gives a transparent projection rule, a linear identified set, closed-form coordinate bounds, and linear-program bounds for means.
Discrete-metric scoring is the known fixed-prize rule with closed-form bounds, and Manhattan distance has an exact identified set with sharp semi-analytical bounds.
A design comparison over these three rules finds no universal ranking: the most informative rule depends on the inferential objective and the design regime.
\end{abstract}
